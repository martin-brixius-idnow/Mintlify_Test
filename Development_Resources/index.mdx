## Setup and integration

### API credentials

Create and manage API credentials, assign permissions, and handle credentials securely.

[Learn more about API credentials](/docs/development-resources/api-credentials)

### Manage webhooks

Set up HTTPS webhooks to receive asynchronous updates about flows and steps, with automatic retries and idempotent processing using `event_id`.

[Learn more about managing webhooks](/docs/development-resources/webhooks-manage)

### Secure webhooks

Secure webhooks using API tokens, optional JWT signatures, IP allowlisting, TLS 1.2+, and standard security headers.

[Learn more about webhook security](/docs/development-resources/webhooks-secure)

### Error handling

Learn how errors are structured, returned, and handled.

[Learn more about error handling](/docs/development-resources/error-handling)
