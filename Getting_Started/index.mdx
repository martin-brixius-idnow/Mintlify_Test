## Quickstart in 3 steps

<Steps>
  <li>
    Create credentials and a sandbox project.<br />
    Go to <a href="/Development_Resources/Manage">Development Resources → API Credentials</a> and <a href="/Getting_Started/Test_Account">Get Started → Test Account</a>.
  </li>
  <li>
    Configure access and webhooks.<br />
    Set roles in <a href="/Getting_Started/Access_Management">Access Management</a> and secure callbacks in <a href="/Development_Resources/Secure">Webhooks → Secure</a>.
  </li>
  <li>
    Execute your first flow and fetch results.<br />
    <a href="/Integration/Execute_Flows">Execute Flows</a> → <a href="/Integration/Get_Flow_Status">Get Flow Status</a> → <a href="/Integration/Get_Datablocks">Get DataBlocks</a>.
  </li>
</Steps>
