# Integrate

Everything you need to call our APIs, run workflows, and consume results — fast and secure.

<Callout type="info">
Start with credentials, then execute a flow and fetch the results. Use the API Explorer to try requests in your browser.
</Callout>

## Quickstart

<Steps>
  <li>
    Get API credentials<br />
    Create keys in <a href="/Development_Resources/Manage">Development Resources → API Credentials</a>.
  </li>
  <li>
    Execute your first flow<br />
    Send a request to <a href="/Integration/Execute_Flows">Execute Flows</a> and capture the flow ID.
  </li>
  <li>
    Track and retrieve results<br />
    Check <a href="/Integration/Get_Flow_Status">Get Flow Status</a> and fetch normalized outputs via <a href="/Integration/Get_Datablocks">Get DataBlocks</a>.
  </li>
</Steps>
